## Описание проекта
Предоставлены данные о свойствах молекул и их биологической активности против вируса гриппа. Задача — построить регрессионные модели, способные предсказывать эффективность новых соединений. В рамках соревнования нужно предсказать три показателя:

- IC50 (mM) — концентрация, при которой вещество подавляет 50% активности вируса
- CC50 (mM) — концентрация, при которой вещество токсично для 50% клеток
- SI (Selectivity Index) — индекс селективности

### Признаки

В изначальном датасете представлено 214 признаков:
-  Общие молекулярные свойства
- Электронные и зарядовые характеристики
- Топологические и структурные дескрипторы
- Поверхностные дескрипторы (VSA)
- Функциональные группы (fr_*)
- Специфические дескрипторы

Подробное описание данных -- на странице соревнования [ChemAI: Predict the Cure](https://www.kaggle.com/competitions/chem-ai-predict-the-cure/data)
